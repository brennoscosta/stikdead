# MARCA
**STIKDEAD** — *Lute. Morra. Evolua.*
Logo: pincelada bruta, STIK branco + DEAD vermelho sangrando. Emblema circular: ceifador sobre o caído.
**Regra de ouro:** preto domina, branco corta, vermelho sangra. Ouro é recompensa, roxo é vazio, azul é gelo. Nenhuma outra cor lidera.
Tom: sombrio com humor seco. Violência estilizada de tinta, nunca gore realista.
